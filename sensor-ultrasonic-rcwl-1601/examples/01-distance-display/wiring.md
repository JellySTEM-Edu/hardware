# Wiring — Example 01: Distance Display

This example uses the RCWL-1601 connected to P13 (Trig) and P14 (Echo) on the JellySTEM Shield.

## Connection table

| Sensor pin | Shield pin |
|------------|------------|
| GND        | GND        |
| VCC        | 3V3        |
| Trig       | P13        |
| Echo       | P14        |

## Will this example work on these pins?

Yes. P13 and P14 are free GPIO pins with no conflicts in this example. The code sets up Trig on P13 and Echo on P14 exactly as wired.
