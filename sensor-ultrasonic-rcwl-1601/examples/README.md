# Examples — RCWL-1601 Ultrasonic Sensor

Three example programs for the RCWL-1601 using the JellySTEM extension. Each example folder contains full setup instructions, block screenshots, JavaScript code, wiring details, and a ready-to-flash `.hex` file.

- **01-distance-display** — Reads distance in cm and shows it on the micro:bit display, updating every half second. Confirms the sensor is working.
- **02-proximity-alert** — Plays a buzzing alarm and lights up the micro:bit LEDs when an object comes within 20 cm. Gets faster as the object gets closer.
- **03-theremin** — Move your hand over the sensor to control the pitch of a musical note in real time, like a theremin instrument.
