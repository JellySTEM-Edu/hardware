# Wiring — Example 03: Theremin

This example uses the RCWL-1601 connected to P13 (Trig) and P14 (Echo) on the JellySTEM Shield. Sound uses the micro:bit V2 built-in speaker.

## Connection table

| Sensor pin | Shield pin |
|------------|------------|
| GND        | GND        |
| VCC        | 3V3        |
| Trig       | P13        |
| Echo       | P14        |

## Will this example work on these pins?

Yes. P13 and P14 are free GPIO with no conflicts. The theremin works best when the sensor is placed face-up on a flat surface with clear space above it — avoid placing other objects nearby that could reflect false echoes.
