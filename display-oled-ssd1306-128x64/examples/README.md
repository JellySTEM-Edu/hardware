# Examples — OLED SSD1306 128×64

Three example programs for the OLED display using the JellySTEM extension. Each example folder contains full setup instructions, block screenshots, JavaScript code, and a ready-to-flash `.hex` file.

- **01-counter** — Press A to count up, B to count down. Tests buttons and display updates.
- **02-bouncing-ball** — A filled circle bounces around the screen using the circle and clear blocks.
- **03-reaction-timer** — A circle appears after a random delay. Press A when you see it and your reaction time shows on screen with a rating.
