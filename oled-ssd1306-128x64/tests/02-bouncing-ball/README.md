# Test 02 — Bouncing Ball

A filled circle bounces around the screen, reflecting off all four edges.

## How to flash

1. Download `02-bouncing-ball.hex`
2. Plug your micro:bit into your computer via USB
3. Drag and drop the `.hex` file onto the `MICROBIT` drive
4. Wait for the micro:bit to restart

## What you should see

A solid circle moving continuously around the screen, bouncing off every edge. No buttons needed — it runs on its own.
