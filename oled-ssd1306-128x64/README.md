# OLED SSD1306 128×64

0.96" I²C OLED display used with JellySTEM projects.

## Specifications

| Property     | Value                   |
| ------------ | ----------------------- |
| Module       | GME12864-11-12-13 V3.22 |
| Display Size | 0.96 inch               |
| Resolution   | 128 × 64                |
| Interface    | I²C                     |
| Drive IC     | SSD1306                 |
| Address      | 0x3C                    |
| Voltage      | 3.3 V–5 V               |

## Verified Configuration

* BBC micro:bit V2
* JellySTEM Shield
* MakeCode
* OLED12864_I2C extension

## Wiring

| OLED | Shield |
| ---- | ------ |
| GND  | GND    |
| VCC  | 3V3    |
| SDA  | P20    |
| SCL  | P19    |
